# 汐汐公主城堡防御

一个使用 Godot 4.7 制作的儿童向魔法少女放置塔防游戏。

游戏采用明亮的梦幻城堡与公主魔法主题。玩家在花园地图上放置不同能力的魔法塔，阻止捣蛋鬼靠近城堡的水晶心。

## 当前版本

当前版本已经完成从旧地下城主题到魔法少女主题的整体更新，并清理了旧版地下城图片、音乐、音效包及遗留代码引用。

已实现：

- 魔法少女主题标题界面、游戏地图、角色、敌人和界面素材。
- 7 种可选择的魔法塔和 10 个固定塔位。
- 自动索敌、发射魔法弹、金币奖励和塔升级。
- 减速、持续伤害和破甲三种攻击效果。
- 消耗星光能量释放全屏“星雨”技能。
- 暂停、x1/x2 速度切换和背景音乐开关。
- 自定义魔法塔名称、初始金币、生命、能量和起始波次。
- Windows 一键打包和已验证的本地运行版本。

当前波次会持续增强并自动开始，暂时没有正式的通关结算和长期进度存档。

## 直接运行源码

项目目录已经放置 Godot 4.7 时：

1. 双击 `play_game.bat` 直接运行游戏。
2. 双击 `open_godot.bat` 打开 Godot 编辑器。
3. 在标题界面点击“开始游戏”。
4. 选择底部魔法塔，再点击地图上的空塔位进行建造。
5. 点击已经建造的塔，可以查看属性并升级。

也可以手动运行：

```bat
tools\godot\Godot_v4.7-stable_win64.exe --path .
```

## 一键打包 Windows 游戏

双击项目根目录的 `一键打包游戏.bat`。脚本会自动导入资源并将游戏打包到 `output/`：

- `汐汐公主城堡防御.exe`：双击启动游戏。
- `汐汐公主城堡防御.pck`：游戏数据文件。

EXE 和 PCK 必须保持同名并放在同一个文件夹。复制给其他 Windows 电脑时，需要同时复制这两个文件，对方不需要安装 Godot。

一键打包脚本使用项目本地的：

```text
tools/godot/Godot_v4.7-stable_win64_console.exe
```

## GitHub 与本地留档

以下内容已在 `.gitignore` 中排除，不会上传到 GitHub：

- `.godot/`：Godot 自动生成的导入缓存。
- `tools/godot/`：本地 Godot 引擎，约 170 MB。
- `output/`：已经打包的 EXE 和 PCK。

因此：

- GitHub 用于保存项目源码、美术、音频、配置和打包脚本。
- 从 GitHub 重新克隆后，需要把 Godot 4.7 放回 `tools/godot/` 才能使用一键打包。
- 本地资料库如果要保存“拿来就能玩”的版本，请额外复制整个 `output/` 文件夹。
- 若以后希望通过 GitHub 分发游戏，建议把 EXE 和 PCK 上传到 GitHub Release，而不是提交到源码仓库。

## 项目目录

- `scenes/`：标题菜单和主游戏场景。
- `scripts/`：游戏、菜单、塔、敌人、子弹、音频及设置逻辑。
- `assets/images/menu/`：标题界面运行素材。
- `assets/images/xiaomox/`：魔法少女塔防运行素材。
- `assets/audio/`：当前主题音乐、魔法音效及音频说明。
- `data/towers.json`：魔法塔数值配置。
- `data/waves.json`：预留的波次配置，目前实际刷怪逻辑仍在 `Game.gd` 中。
- `img/`：魔法少女和菜单的原始拆分美术。
- `design/`：界面设计概念稿。
- `export_presets.cfg`：Windows 导出配置。

## 音频说明

当前运行版本只保留魔法少女主题背景音乐和 `sfx_magic` 音效。详细清单见 `assets/audio/ATTRIBUTION.md`。

仓库中尚未记录这些当前音频的外部作者或授权来源；若未来公开发布，应先补充并确认授权信息。
